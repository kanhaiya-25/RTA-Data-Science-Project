# Road Traffic Accident Data Analysis

## Problem Statement
Analyze accident severity and identify key factors affecting road accidents.

## Dataset
RTA Dataset with 12,000+ records.

## Steps Performed
- Data cleaning
- Feature engineering
- EDA
- API integration
- Modeling (classification)
- Power BI dashboard

## Tools Used
- R
- Power BI
- GitHub

## Conclusion
Accident severity is influenced by driver behavior, weather, and road conditions.
